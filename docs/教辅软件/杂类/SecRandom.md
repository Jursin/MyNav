- [X] 抽单人/抽多人/抽小组/抽性别 等抽取方式
- [X] 结果语音提示功能
- [X] 可开关的历史记录
- [X] 带时间戳的详细抽取记录
- [X] 多班级名单管理
- [X] 独立抽取与设置界面
- [X] 便捷悬浮窗设计
- [X] 自动清除抽取记录
- [X] 抽取动态权重系统
- [X] 公平抽取的概率可视化
- [X] 开机自启动支持
- [X] 现代化UI设计
