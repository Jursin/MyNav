## 功能
- [x] 随机抽取
- [x] 可视化配置菜单
- [x] 从外部读取名单
  - 示例：（0=男，1=女）
    ```csv
    name,sex,no
    example,0,1
    ```
- [x] 特殊点名规则
- [x] 支持非二元性别
- [X] 同时抽选多个
- [X] 悬浮窗（点击展开主界面）
- [ ] 概率内定
- [ ] 软件内更新
- [ ] 播报抽选结果
- [ ] 手机遥控抽选
